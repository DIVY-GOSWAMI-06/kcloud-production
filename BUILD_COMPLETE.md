## 🔧 Customization Quick Reference

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  gold: "#your-color",
  darkGray: "#your-color"
}
```

### Change Fonts
Edit `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT..." />
```

### Update Content
Edit component files:
- `Services.jsx` - Services list
- `Portfolio.jsx` - Portfolio items
- `About.jsx` - About content
- `Footer.jsx` - Contact info

### Replace Images
Search for placeholder gradients and replace with:
```javascript
<img src="/path/to/image.jpg" alt="Description" />
```
