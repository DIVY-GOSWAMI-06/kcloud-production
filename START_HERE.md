## ⚡ STEP 1: INSTALL (2-3 minutes)

Open Command Prompt/PowerShell and run:

```bash
cd d:\Mock
npm install
```

**What it does:** Downloads all necessary packages (React, Tailwind, Framer Motion, etc.)

**You'll see:** Progress bars as packages are installed

**When done:** You'll see ✅ completion message

---

## 🔥 STEP 2: RUN (30 seconds)

In the same terminal, run:

```bash
npm run dev
```

**What it does:** Starts your development server

**You'll see:** 
- `VITE v5.0.8 ready in 234 ms`
- `Local: http://localhost:3000`
- Browser may auto-open

**If not auto-opened:** Go to http://localhost:3000 manually

## 🆘 STUCK?

### Port 3000 in use?
```bash
npm run dev -- --port 3001
```

### npm install fails?
```bash
npm cache clean --force
npm install
```

### Browser won't load?
- Check terminal for errors
- Try http://localhost:3000 manually
- Hard refresh: Ctrl+Shift+R

### Changes not showing?
- Save file: Ctrl+S
- Check console (F12)
- Restart server: Ctrl+C, npm run dev
